# 🕊 BIRD2 Extension Pack

</div>

A convenience extension pack for BIRD2 users.

This pack installs both:

- `birdcc.bird2-lsp` (BIRD2 LSP)
- `birdcc.vscode-bird2-conf` (BIRD2 syntax highlighting)

## Why this pack

- One-click setup for full BIRD2 editing experience.
- Keeps syntax highlighting and language tooling decoupled.
- Makes onboarding simpler for new users.

## Included extensions

### BIRD2 LSP

Provides diagnostics, hover, formatting, validation, and language server features.

### BIRD2 Configuration

Provides TextMate grammar based syntax highlighting for BIRD2 config files.

## Requirements

- VS Code 1.80 or newer.
